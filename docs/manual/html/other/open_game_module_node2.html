Introduction

Open Game Module is a open source expansion card for the Colecovision. This module is Super Game Module (SGM) compatable. This means all SGM games will run when this module is used with a Colecovision. Includes the same 32KB of RAM. The sound is expanded with a Yamaha YMZ284 instead of the original AY chip.
